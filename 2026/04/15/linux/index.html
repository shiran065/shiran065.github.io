linux

ls -a查看隐藏文件
-l竖着排列
-lh显示文件大小
cd切换工作目录
cd .是当前目录
cd ..是上一级的目录
cd ~是回到home目录
cd -回到上一工作目录
pwd显示工作目录
mkdir创建新目录
mkdir -p创建多层级文件夹
touch创建文件
cat查看文件内容
more也是查看文件内容，不过可以一页页看
cp [-r] 参数1 参数2 复制文件或文件夹，文件夹要带-r
mv 参数1 参数2 移动文件或文件夹 或改名
rm [-r]删除文件或文件夹
（可以使用通配符text表示删除以text开头的内容）text text
which 要查找的命令
find 起始路径 -name “被查找的文件”（也可以使用通配符）
find 起始路径 -size +或-[文件大小]
su - root切换管理员模式，exit命令退出
grep [-n] “关键字” 文件路径
用与利用关键字过滤文件行，输出有关键字的行
wc [-l -w -c -m] 文件路径
-c统计字节数量，-m统计字符数量，-l统计行数，-w统计单词数量
管道符｜将左边命令的结果作为右边命令的输入
echo “输出的内容”
echo ‘命令’
重定向符 > 指将左边的内容覆盖写到右边 >> 指将左边的内容追加到右边
tail [-f -num]表示查看文件尾部内容，-f表示持续跟踪
vim 文件名 可进入vim编辑器，i键进入输入模式，esc退回命令模式，按：键进入底线命令模式
groupadd 创建用户组
groupdel 删除用户组
useradd 创建用户 userdel[-r]删除用户 不加-r会保留home目录
usermod-aG 用户组 用户名，将用户加入用户组
getent passwd显示系统中有哪些用户
chmod [-R] u=rwx g=rwx o=rwx可以修改文件，文件夹信息 -R直对文件夹中的内容进行同样的操作 u代表用户，g代表用户组，o代表其他用户
chown [-R] [用户] [:] [用户组] 文件或文件夹
可以修改文件，文件夹的所属用户和用户组，只适用于root用户
ctrl+C强制停止程序
ctrl+D退出登录
history查看历史命令
!加字符可以查找对应字符开头的命令并执行
ctrl+r输入内容匹配历史命令
ctrl+a跳到命令开头
+e跳到结尾 +左键左跳一个 单词 右键右跳
ctrl+l清空
yum[-y] [install |remove |search]
ln -s 参数1 参数2 参数1是目标文件或文件夹 参数2是要生成快捷方式的位置
data [-d] [+格式化字符]用于查看系统时间 -d用于计算日期
ping [-c num]检查指定的网络服务器是否是可联通状态
ps [-e -f]可以查看进程，-e显示全部进程 -f显示进程全部信息 配合grep查看固定的进程
kill [-9] “进程ID” 关闭进程
top命令查看CPU，内存使用情况
df [-h]命令，查看磁盘使用情况
sar [-n] DEV 查看网络的相关统计
