sql

SQL通用语法
1.SQL语句以分号结尾。
2.可以使用空格/缩进。
2.MYSQL语句不分大小写，关键字用大写。

DDL(数据定义语言,用来定义数据库对象)
1.数据库操作
查询所有数据库show databases;
查询当前数据库select databases();
创建数据库create databases [if not exists] 数据库名 [default charset 字符集] [collate 排序规则];
删除数据库drop database [if exists] 数据库名;
use 数据库名;

2.表结构操作
展示所有表show tables;
展示表结构desc 表名；
展示创建的表格show create table 表名；
创建表格create table 表名（
字段 字段类型[comment 字段注释]，
……
）[comment 表注释];
添加\修改字段alter table 表名 add/modify 字段名 类型 [comment注释信息];
改字段名alter table 表名 旧字段名 新字段名 类型;
删字段alter table 表名 drop 字段名;
修改表alter table 表名 rename to 新表名;
删除表drop table [if exists]表名;
删除重建表truncate table 表名;
数值类型tinyint smallint mediumint int或integer bigint float double decimal
字符串类型char verchar tinyblob tingtext blob text mediumblob longblob longtext

DML(数据操作语言，用来对数据库的数据进行增删改)
给指定字段添加数据insert into 表名 (字段1,字段, …) values (值1，值2…);
给全部字段添加数据insert into 表名 values (值1，值2…);
修改数据update 表名 set 字段名1=值1，…[where 条件];
删除数据delete from 表名 [where 条件]

DQL(数据查询语言，用来查询数据库中的表的记录)
基础语法
select 字段列表 from 表名列表 where 条件列表 group by 分组字段列表 having 分组后条件列表 order by 排序字段列表 limit 分页参数;
语法说明
select 字段列表
from 表名列表
where 条件列表
group by 分组字段列表
having 分组后条件列表
order by 排序字段列表
limit 分页参数;

DCL(数据控制语言，用来创建数据库用户、控制数据库的访问权限)

  1. 管理用户
    查询用户：
    use mysql;
    select * from user;
    创建用户：create user ‘用户名‘@’主机名’ identified by ‘密码’;
    修改用户密码：alter user ‘用户名‘@’主机名’ identified with mysql_native_password by ‘新密码’;
    删除用户：drop user ‘用户名‘@’主机名’;

  2. 权限控制
    查询权限：show grants for ‘用户名‘@’主机名’;
    授予权限：grant 权限列表 on 数据库名.表名 to ‘用户名‘@’主机名’;
    撤销权限：revoke 权限列表 on 数据库名.表名 from ‘用户名‘@’主机名’;

函数
1.字符串函数
concat(s1,s2,…sn) 拼接多个字符串为一个整体 select concat(‘hello’, ‘ ‘, ‘mysql’); hello mysql
lower(str) 将字符串全部转为小写 select lower(‘HELLO MySQL’); hello mysql
upper(str) 将字符串全部转为大写 select upper(‘hello mysql’); HELLO MYSQL
lpad(str,n,pad) 左填充：用 pad 把 str 补到 n 个字符长度 select lpad(‘123’, 5, ‘0’); 00123
rpad(str,n,pad) 右填充：用 pad 把 str 补到 n 个字符长度 select rpad(‘123’, 5, ‘‘); 123*
trim(str) 去除字符串首尾的空格（中间空格保留） select trim(‘ hello mysql ‘); hello mysql
substring(str,start,len) 从 start 位置开始，截取 len 个字符（SQL 字符串从 1 开始计数） select substring(‘hello mysql’, 7, 5); mysql

2.数值函数
ceil(x) 向上取整（只要有小数就进 1） select ceil(1.2); /select ceil(-1.2); 2 / -1
floor(x) 向下取整（直接舍弃小数部分） select floor(1.9);/ select floor(-1.9); 1 / -2
mod(x,y) 取模（求 x 除以 y 的余数） select mod(10, 3); /select mod(-10, 3); 1 / -1
rand() 生成 0~1 之间的随机浮点数 select rand(); （每次运行结果不同，如 0.789）
round(x,y) 四舍五入，保留 y 位小数 select round(3.14159, 2);/ select round(3.5); 3.14 / 4

3.日期函数
curdate() 返回当前日期（格式：yyyy-mm-dd） select curdate(); （如 2026-04-21）
curtime() 返回当前时间（格式：hh:mm:ss） select curtime(); （如 15:30:00）
now() 返回当前日期 + 时间 select now(); （如 2026-04-21 15:30:00）
year(date) 获取日期的年份 select year(‘2026-04-21’); 2026
month(date) 获取日期的月份 select month(‘2026-04-21’); 4
day(date) 获取日期的天数 select day(‘2026-04-21’); 21
date_add(date, interval expr type) 给日期加上指定时间间隔（type 支持 day/month/year/hour 等） select date_add(‘2026-04-21’, interval 3 month); 2026-07-21
datediff(date1, date2) 计算两个日期的天数差（date1 - date2） select datediff(‘2026-05-01’, ‘2026-04-21’);

4.流程函数
if(value, t, f) 条件判断：value 为真返回 t，否则返回 f select if(10 > 5, ‘是’, ‘否’); 是
ifnull(value1, value2) 空值处理：value1 不为空返回 value1，否则返回 value2 select ifnull(null, ‘默认值’); 默认值
case when 多条件分支（两种用法：等值判断 / 范围判断） select case when score >= 90 then ‘优秀’ when score >= 60 then ‘及格’ else ‘不及格’ end from student;

约束
1.非空约束 not null
2.唯一约束 unique
3.主键约束 primary key(一张表只能有一个主键) auto_increment 自动增长
4.默认约束 default
5.检查约束 check
6.外键约束 foreing key
创建表时添加外键：
create table 表名(
字段名 数据类型,
…
[constraint] [外键名称] foreign key (外键字段名) references 主表 (主表列名)
);

创建表后添加外键: alter table 表名 add constraint 外键名称 foreign key (外键字段名) references 主表 (主表列名);

no action 当在父表中删除 / 更新对应记录时，首先检查该记录是否有对应外键，如果有则不允许删除 / 更新。（与 restrict 一致）
restrict 当在父表中删除 / 更新对应记录时，首先检查该记录是否有对应外键，如果有则不允许删除 / 更新。（与 no action 一致）
cascade 当在父表中删除 / 更新对应记录时，首先检查该记录是否有对应外键，如果有，则也删除 / 更新外键在子表中的记录。
set null 当在父表中删除对应记录时，首先检查该记录是否有对应外键，如果有则设置子表中该外键值为 null（这就要求该外键允许取 null）。
set default 父表有变更时，子表将外键列设置成一个默认的值（innodb 不支持）
示例
alter table emp add constraint fk_emp_dept_id foreign key (dept_id) references dept(id) (on update cascade on delete cascade) ;
alter table emp add constraint fk_emp_dept_id foreign key (dept_id) references dept(id) (on update set null on delete set null) ;

多表查询
1.内连接(查询的是两表之间的交集)
隐式内连接 select 字段列表 from 表1, 表2 where 条件 …;
显式内连接 select 字段列表 from 表1 [inner] join 表2 on 连接条件 …;
示例
select emp.name , dept.name from emp , dept where emp.dept_id = dept.id ;
select e.name,d.name from emp e , dept d where e.dept_id = d.id;（取别名）

2.外连接
左外连接/右外连接(一边表的所有数据和两表之间的交集)
select 字段列表 from 表1 left/right [outer] join 表2 on 条件…;

3.自连接
select 字段列表 from 表A 别名A join 表A 别名B on 条件…;
示例
select a.name b.name from emp a , emp b where a.managerid = b.id;

4.联合查询
select 字段列表 from 表1
union [all]
select 字段列表 from 表2;

5.子查询
select * from t1 where column1 = (select column1 from t2);

事务
设置事务提交方式:select @@autocommit; 查看当前 MySQL 是否开启自动提交模式（1 表示开启，0 表示关闭）。
set @@autocommit = 0; 关闭自动提交，后续的增删改操作需要手动执行 commit 才会永久生效。
commit; 提交事务，将事务内的所有修改永久写入数据库。
rollback; 回滚事务，撤销事务内所有未提交的修改。
开启事务:start transaction 或 begin;
提交事务:commit;
回滚事务:rollback;
事务四大特性:原子性，一致性，隔离性，持久性
